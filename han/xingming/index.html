<u>帜和语</u>本土姓名

# 帜和语本土姓名

帜和语并没有成型的架空设定，但有「本土」人名的设定。

# 结构

帜和语人名的结构是名+姓，先名后姓。

# 名

由普通名词或动词加人名后缀构成。人名后缀主要是*-ina*（女名）和*-ano*（男名），偶有*-ana*（女名）和*-ino*（男名）。

例：

  • pase（照亮的） -> Pasano（帕萨诺）
  • veroda（真实） -> Verodano（韦洛达诺）
  • faha（云） -> Fahano（法哈诺）
  • vere（真的） -> Verina（韦利那）
  • more（海） -> Morina（莫利那）
  • jukome（希望）-> Jukomina（尤科米那）
  • jehe（好的）-> Jehana（耶哈那）
  • mes（想） -> Mesino（梅西诺）

# 姓

起源于父称或母称，由某个祖先的名字加姓氏缀派生而来。常见姓氏缀及示例：

姓氏缀 来源 例子
-isin sine 子（旧，塞源） Pasano -> Pasanisin（帕萨尼辛）
-ikot kote 孩子（旧，立源） Jehana -> Jehanikot（耶哈尼科特）
-el elh 子（立准语） Morina -> Morinel（莫里内勒）
-ic -ic- 指小词缀（塞源） Stelina -> Stelinic（斯忒里尼奇）
-ov -ov- 集合词缀（塞源） Karano -> Karanov（卡拉诺夫）
sna- sine a ……之子 Gorano -> snaGorano（斯那戈拉诺）
kota- kote a ……的孩子 Jonina -> kotaJonina（科塔尧尼那）
eli- elh i' ……之子（立准语） Sofana -> eliSofana（艾里索法那）

带姓氏前缀的姓氏在大写上有两种不同习惯：

新式：姓氏缀无论前缀后缀，姓氏缀均不大写（不加专名号），祖先名即使加了前缀仍旧大写（加专名号），如Pasanisin、snaGorano。

旧式：无论首字母是不是姓氏缀，均首字母大写。如Pasanisin、Snagorano。

# 缩写

# 首字母缩写

使用首字母缩写时取名和姓的大写字母加**'**。如：

  • Verodano Pasanisin -> V'P'（）
  • Kulina Morinel -> K'M'（）

对于带前缀的姓氏，与大小写的两种方式对应地，有两种缩写方式：

  • Jukomina snaGorano -> J'G'
  • Jukomina Snagorano -> J'S'

# 辅音式缩写

名字词干省略元音，词缀也各做省略。常用于签名。

词缀 缩写 例子
-ina -i Verina -> Vri'（）
-ano -a Verodano -> Vrda'（）
-ana -aa Jehana -> Jhaa'（）
-ino -io Mesino -> Msio'（）
-isin -s Pasanisin -> Psas'（）
-ikot -k Jehanikot -> Jhak'（）
-el elh 子（立准语） Morinel -> Mril'（）
-ic -ic- 指小词缀（塞源） Stelinic -> Stlic'（）
-ov -ov- 集合词缀（塞源） Karanov -> Krav'（）
sna- sine a ……之子 snaGorano -> sGra'（）
kota- kote a ……的孩子 kotaJonina -> kJni'（）
eli- elh i' ……之子（立准语） eliSofana -> lSfaa'（）

写全名时名、姓分别大写（加专名号）并省去中间的空格和**'**。如：VrdaPsas'（）

# 翻译

帜和语人名翻译为汉语时，一般采用音译；但有时也可以意译，如：

  • faha（云）, vike（高）-> Fahano Vikanic - 高云

  • cule（规定）, stela（星星）-> Culano Stelinic - 星律

# 下划线缩短

可用于连续专名号。[xxx]{.u}由markdown-it-attrs和markdown-it-bracketed-spans插件实现。例：

先拂市海西区执鲁街道

商弦市景公河东街上河湾路口

<u>先拂市</u><u>海西区</u><u>执鲁街道</u>

[商弦市]{.u}[景公河东街]{.u}[上河湾路口]{.u}

# 无边框表格

去掉边框，首行加粗改为倾斜，行高调小。可用于莱比锡标注系统（Leipzig Glossing）。markdown-it-abbr插件可自动识别已定义的缩写，且自动设为小型大写；也可以使用[xxx]{.ab}来手动设定小型大写。例：

m-i t-e zes.
I-NOM you-ACC love.IMPF
'I love you.' (ACC = accusative)
*[NOM]: Nominative case
<!--*[ACC]: Accusative case-->
*[IMPF]: Imperfective aspect

| m-i   | t-e   | zes. |
| :---- | :---- | :---- |
| I-NOM | you-[ACC]{.ab} | love.IMPF |
| 'I love you.' ([ACC]{.ab} = accusative) |||
